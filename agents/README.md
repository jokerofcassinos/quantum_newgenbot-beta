# 🤖 AGENTS DIRECTORY - FOREX QUANTUM BOT

## 📖 COMO USAR ESTE DIRETÓRIO

Este diretório contém **definições de agentes** que você pode criar individualmente usando o comando da CLI do Qwen Code.

### Passo a Passo para Criar um Agente:

1. **Abra o arquivo .md do agente** (ex: `market-researcher.md`)
2. **Copie o conteúdo** da seção "PROMPT DO AGENTE"
3. **Use o comando da CLI:**
   ```
   /agent create market-researcher
   ```
4. **Cole o prompt** quando solicitado
5. **Repita** para cada agente
